## About Project
This project improves the security of a PHP application.  
It protects the application from common attacks and manages user access safely.

## What I Did
- Used prepared statements to prevent SQL injection  
- Added form validation for user inputs  
- Implemented secure login and registration  
- Added user roles (Admin and User)

## Technologies Used
- PHP  
- MySQL  
- HTML  
- CSS  
- XAMPP  


## Security Features
- Prepared statements (MySQLi)  
- Password hashing using `password_hash()`  
- Role-based access control  
- Server-side validation  

## How to Run
1. Start XAMPP (Apache & MySQL)
2. Create database `secure_app`
3. Place project in `htdocs`
4. Open browser:http://localhost/security-task4/register.php
